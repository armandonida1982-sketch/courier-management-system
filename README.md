# 🚚 Courier Management System

A full-stack courier application designed for logistics and delivery operations.

---

## 🚀 Features

### 🧑‍💼 Admin (Web App)
- Dashboard monitoring
- Courier & sorter management
- Order assignment
- Attendance tracking
- Report & analytics

### 🏢 Client (Web App)
- Upload order (CSV / Excel)
- Track shipment status
- Dashboard monitoring

### 📦 Courier (Mobile App)
- Pickup courier
- Delivery courier
- Scan barcode system
- Check-in / Check-out attendance
- Earnings tracking

### 🏭 Sorter (Mobile App)
- Package sorting system
- Scan & validation
- Work tracking

---

## 🧩 Tech Stack
- Frontend: FlutterFlow
- Backend: Supabase
- Database: PostgreSQL
- Storage: Supabase Storage

---

## ⚙️ System Roles
- Admin
- Client
- Pickup Courier
- Delivery Courier
- Sorter

---

## 📈 Project Scope
- Multi-role system
- Real-time data sync
- Mobile & Web integration
- Scalable backend
